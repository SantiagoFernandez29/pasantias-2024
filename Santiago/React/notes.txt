*React solo renderiza elementos, por lo que previamente hay que crearlos.


*Crear un Root como base de todo el Ã¡rbol de elementos que se vaya a formar.
*Creo un elemento, al que le paso el tipo de elemento, los atributos que debe cumplir, y lo que envuelve dicho elemento.
*Indicar que se quiere renderizar dentro.
*Solo se puede renderizar un elemento, por lo que si por ejemplo tenemos 3 botones que renderizar, entonces lo mejor es envolverlos en un div y renderizar este div, que envuelve un array con cada uno de los botones.
*Si no quiero renderizar ese div, entonces puedo usar el elemento React.fragment, que sirve para renderizar los elementos que envuelva sin que el fragment se renderice.


*JSX permite crear elementos de forma declarativa, dejando de lado la forma imperativa que arrastra JS.
